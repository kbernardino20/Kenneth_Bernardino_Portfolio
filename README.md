## Mobile optimisation

This build includes a mobile-first layout for 320px and above, a touch-friendly navigation drawer, 44px minimum tap targets, safe-area support, one-column work samples and certificate layouts on phones, responsive case-study galleries, and reduced hover behaviour on touch devices.

## Homepage slideshow update
The homepage uses a four-image cinematic slideshow with previous/next controls, navigation dots, automatic rotation, reduced-motion support, and responsive image positioning. The Contact section includes a WeChat username button that copies `@kenbernardino`.


## Final light-mode design

The homepage uses light mode by default, includes a responsive three-photo hero collage, and uses consistent bordered contact cards for Email, LinkedIn, GitHub, Facebook and WeChat. Dark mode remains available through the theme toggle.

## Latest design update
The homepage professional highlights are presented as a compact, non-clickable information strip with clear separators rather than interactive cards.
